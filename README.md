# Internet-safety
Теоретический справочник по безопасности в интернете: как защитить свои устройства, сервисы, данные и финансы от мошенников. Узнайте, как не стать жертвой обмана и правильно хранить документы. А если защита дала сбой - узнать что делать с алгоритмом действий для минимизации ущерба и возврата доступа к аккаунтам.
